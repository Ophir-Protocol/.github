## Ophir Protocol

Open protocol for AI agents to discover, negotiate, and transact autonomously.

```json
{
  "mcpServers": {
    "ophir": {
      "command": "npx",
      "args": ["@ophirai/mcp-server"]
    }
  }
}
```
